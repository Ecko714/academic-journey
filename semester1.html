  • ASU 101-CSE "The ASU Experience"

    • Course Description

    "ASU 101 is a required, one-credit course designed to introduce all new first-time ASU students to the unique elements, culture, challenges, and opportunities of ASU and Fulton Schools of Engineering."

    Ahh, you never forget your first class... ASU 101 was a mandatory class for every freshman. We learned the basics of being a student at ASU: communicating with just about everyone (faculty, advisors, even our peers). We discussed important things like understanding syllabi, creating your own schedules, and navigating social interactions (I think a lot of students need a refresher on this one #ThanksCovid #AllTeaAllShade). Not much to say about this class.

  • CSE 110 - Principles of Programming

    • Course Description

    "This course is an introductory course to programming using Java language... The course concentrates on programming concepts, problem solving, and program design."

    Alright, an actual class. ASU starts Computer Science majors off with Java (I want to say there is also a python or C++ class that you can start with but this class was on my major map so I'm stickin' with it). You learn fundamental concepts like primitive data types, variables and functions, as well as other Java-specific tools like classes. The class is built for students who have ZERO programming knowledge which I think is really cool. There are things that I've learned in this class that I still utilize to this day! 10/10 would take again.

    (P.S. Professor Yoshi was the best)

  • ENG 101 - First-Year Composition

    • Course Description

    "This course is designed to allow students to learn about the processes of communicating effectively in writing, that is, with purpose and towards specific goals."

    There is a reason I am pursuing a STEM degree. Oh my goodness, this class was a DRAG. I will say, the professor that I had for this class was one of the better professors that I've interacted with. He was very compassionate and was ready to listen to anything you had to say which I appreciate considering the classroom is a space for learning and understanding. I once had a passion for writing but that fire was smothered long ago. Creative pieces and analytic pieces were written in this course. 0/10 would take again.

  • FSE 100 - Introduction to Engineering

    • Course Description

    "Introduces the engineering design process; working in engineering teams; the profession of engineering; engineering models, written and oral technical communication skills.

    Back to the fun stuff! FSE 100! What an interesting class. Here, the class learned how to tackle engineering questions, what it means to be an engineer, and most importantly, how to work in a collaborative environment. The main project was a Lego© robot set that was programmable. We used MATLAB as our IDE which was a first for me. I was always used to Visual Studio Code or Atom as my IDE (for the uninitiated, an IDE is an Integrated Development Environment! It's software that makes writing code ~so~ much easier). Very enjoyable class and I was lucky enough to have a great professor for this class as well.
    Shout out Professor Osburn! (I end up having him for two other classes!)

  • GCU 121 - World Geography

    • Course Description

    "In this class we will:
    - Locate world realms & regions.
    - Identify patterns of distribution of phenomena in these regions.
    - Understand on a broad level the changes that have taken place in these places.
    - Compare and contrast these regions of the world with each other, as well as with the region(s) with which you are most familiar.""

    Although I took this class to mainly fufill a graduation requirement, Geography has always been a subject that I have been interested in. As this was a beginner class, the material was easy to digest and wasn't very overbearing. Assignment-wise we had 4 exams, about a quiz every other week, and a few discussion posts that had a light amount of research go into them. Interesting class, not much more to say.

  • MAT 171 - Precalculus: STEM

    • Course Description

    Students will be able to solve and graph various types of algebraic functions (including polynomial, rational, exponential and logarithmic) as well as apply this knowledge to more complex problems. Students will also be able to use the six basic trigonometric functions and their inverses to solve problems, both applied and theoretical.

    Ohhhhhh boy. And here we begin my mathematics journey. Not excited. Pre-Calculus. In hindsight, the class was not hard. It was your average math class: come to class, learn about a topic, you have a couple of days to finish homework for that topic before you go back to learn about a new thing. It moved at a steady pace from learning about different functions to exploring different ideas like trigonometry with everyone's favorite: the unit circle. Technically, I should've taken this in high school but that's neither here nor there.

  • And that is a wrap on my first semester at ASU! I learned a solid amount. Not too much in the computer science area but I have seven other semesters for that! We'll get there.
